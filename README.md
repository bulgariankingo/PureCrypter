## 🔍 What Is This?

Pure Crypter is a simple crypter/stub architecture that encrypts a PE (Portable Executable) file and decrypts it at runtime in memory.

**Intended use cases:**

- Prevent your own harmless software from being falsely flagged by antivirus engines
- Learn how encryption and runtime payload execution work
- Use in controlled Red Team training environments

---

## ⚙️ Features

- AES-256 encryption for payload hiding
- In-memory execution (no disk trace)
- Simple, auditable stub structure
- Fully open source – **no backdoors, no hidden code**
